# 宠物与主人的宿命关系

一个温暖的小测试，探索你和毛孩子之间的缘分。

## 在线体验

👉 [点击体验](https://hjf0007733-commits.github.io/pet-destiny/)

## 功能介绍

### 宿命解读
根据你和宠物的信息，匹配 6 种宿命关系类型：
- 静默的疗愈者
- 灵魂的镜像
- 温柔的引路人
- 时光的守望者
- 情绪的摆渡人
- 宿命的同行者

### 彩虹桥模式
为已经离开的毛孩子设计的特别版本，用更温柔的文字纪念那段珍贵的陪伴。

### 专属故事
生成一段属于你们的小故事，记录相遇、相伴的点滴。

### 纪念卡片
生成可截图保存的宿命总结卡片。

## 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- 无需后端，单文件即可运行
- 响应式设计，支持移动端

## 本地运行

直接用浏览器打开 `index.html` 即可。
